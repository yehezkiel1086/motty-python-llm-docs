# Langchain LLM Docs

Docs about LLM implementations tools like LangChain

## Requirements

- Python >= 3.10
- pip >= 24.3.1

## Installation

1. Create a virtual environment

    ```sh
    python3 -m venv .venv
    ```

2. Activate

    ```sh
    . .venv/Script/activate
    ```

3. Install requirements

    ```py
    pip install -r requirements.txt
    ```
